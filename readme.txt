BOXFUL(HK)電商出貨小幫手：One-stop e-commerce Fulfillment for WooCommerce

== Description ==
BOXFUL(HK)電商物流為使用WooCommerce的品牌電商提供一站式的電商物流解決方案，透過系統串接自動撈取您的WooCommerce訂單，根據訂單完成高效率電商揀貨出貨，並提供台灣宅配、超商取貨及跨境多國配送等多元物流選項，讓您即使出門在外，電腦不在手邊，也能透過這組外掛程式輕鬆完成電商出貨。

= 為什麼選擇我們? =
- 全自動串接出貨：自動撈取訂單、更新貨態，不再需要手動整理訂單
- 多通路配送：提供宅配、超商取貨、雙北8HR超速配及跨境物流服務
- 雲端訂單庫存管理系統：雲端即時查看訂單及庫存狀況
- 跨通路庫存管理：整合多個不同通路即時更新庫存，避免超賣危機
- 跨境物流整合：整合多個跨境物流商，系統自動選取最適物流，輕鬆賣爆全世界


= 如何使用 =
- 此外掛僅提供BOXFUL(HK)電商物流合作企業戶使用，無需額外安裝費用
- 在使用此外掛前您需要先完成BOXFUL(HK)電商物流帳號開立，詳情請洽BOXFUL(HK)電商物流網站 https://www.boxful.com/fulfillment
- BOXFUL(HK)顧客成功顧問會協助您開立帳號及後續外掛安裝啟動事宜
